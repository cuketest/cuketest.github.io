CukeTest Demo Scripts

All Cucumber.js 3.x sample scripts can work well in CukeTest. To help user start quickly, here provides some demo scripts and demo applications that user can download:

Demo Applications

First, we hosted demo sample web applications on gitub.io, and users can create their web testing scripts against these demo applications:

  • Calculator a web calculator
  • Shopping a small web site simulate online shopping web pages
  • Shopping Cart a small web site simulate adding products to shopping cart, it is independent from the previous sample

Demo Scripts

Correspondingly, the following demo scripts are provided to automate above 3 web applications

  • Calculator script Automate above web Calculator
  • Shopping Script Automate "Shopping" application
  • Shopping-Cart Script Automate "Shopping Cart" application

Besides these scripts, you can find some more demo scripts on CukeTest demo project.

The downloaded script script point directly to the applications hosted on github.io. Improve performance, you can git clone repository from here and find applications in it.

CukeTest演示脚本

所有Cucumber.js 3.x示例脚本在CukeTest中都可以正常工作。为了帮助用户快速启动，这里提供了一些用户可以下载的演示脚本和演示应用程序：

演示应用程序

首先，我们在github.io上托管Web应用程序的演示示例，用户可以针对这些演示应用程序创建其Web测试脚本：

  • Calculator 一个Web计算器
  • Shopping 一个小网站模拟在线购物的网页
  • Shopping Cart 一个小型网站模拟添加产品到购物车，它是独立于上面的样例

演示脚本

相应的，下面的演示脚本提供了自动化以上三个Web应用程序

  • Calculator script 自动化上面的网页计算器
  • Shopping Script 自动化 "Shopping" 应用
  • Shopping-Cart Script 自动化 "Shopping Cart" 应用

除了这些脚本，你还可以在CukeTest 演示 项目上找到更多的演示脚本.

下载的脚本直接指向github.io上托管的应用程序。提高性能，你可以从 这个git仓库 克隆并运行这个Web应用。
